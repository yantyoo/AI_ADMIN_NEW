# Guidelines

## Product Rules

- Keep admin shell structure stable
- Preserve list/detail/modal pattern
- Keep role-based visibility for MASTER and OPERATOR
- Keep loading, empty, error, and modal states

## Visual Rules

- Use the shared theme tokens
- Preserve the existing B2B admin visual tone
- Do not introduce a new color system

## Content Rules

- Keep the copy terse and operational
- Prefer concrete labels over marketing language
