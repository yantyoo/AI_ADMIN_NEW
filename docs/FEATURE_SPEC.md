# XpERP 챗봇 어드민 기능정의서

## 문서 목적
- 본 문서는 로컬 단일 기준 기능정의서다.
- Claude와 Codex는 Notion보다 본 문서를 우선 참조한다.
- 기능 추가, 수정, 삭제는 본 문서에서 먼저 반영한다.
- Notion 반영은 사용자의 명시적 요청이 있을 때만 수행한다.

## 이관 상태
- 상태값: `complete_import`
- 총 CSV 항목 수: `114`
- 현재 로컬 반영 수: `121`
- 정렬 기준: `대메뉴 고정 순서 > ID 오름차순 > Depth1 > Depth2 > Depth3 > 기능명`
- 원본 파일: `docs/FEATURE_SPEC_SOURCE.csv`
