# XpERP Admin Importer

이 플러그인은 `XpERP AI Admin`의 화면 구조를 Figma 캔버스에 빠르게 배치하기 위한 최소 import 도구입니다.

## 사용 방식
1. Figma 데스크톱 앱에서 Plugins > Development > Import plugin from manifest... 선택
2. `figma-plugin/manifest.json` 선택
3. 플러그인 실행

## 생성 결과
- `XpERP Admin Import` 페이지 생성
- `/login`, `/dashboard`, `/content`, `/cache-qa`, `/knowledge`, `/feedback`, `/accounts` 프레임 생성

## 제한
- 이 버전은 화면 구조 배치용입니다.
- 실제 디자인 시스템 컴포넌트 매핑은 다음 단계에서 진행합니다.
